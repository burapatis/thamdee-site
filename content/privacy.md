นโยบายฉบับนี้ใช้กับเว็บไซต์ **thamdee.com** ซึ่งเป็นเว็บไซต์หลักของ Thamdee Ecosystem แอปพลิเคชันและเว็บไซต์ย่อยแต่ละรายการอาจมีนโยบายความเป็นส่วนตัวเฉพาะของตนเอง ซึ่งจะแสดงไว้ในหน้าของโปรเจกต์นั้น ๆ

## ข้อมูลที่เราเก็บ

เว็บไซต์ thamdee.com เป็นเว็บไซต์แบบ Static ที่ **ไม่มีระบบสมัครสมาชิกและไม่เก็บข้อมูลส่วนบุคคลโดยตรง** ข้อมูลที่อาจเกิดขึ้นจากการใช้งานมีดังนี้

- **ข้อมูลการติดต่อ** — เมื่อคุณส่งอีเมลถึงเรา เราจะได้รับชื่อ อีเมล และเนื้อหาที่คุณส่งมา และใช้เพื่อการตอบกลับเท่านั้น
- **ข้อมูลทางเทคนิคพื้นฐาน** — ผู้ให้บริการโฮสติ้ง (เช่น GitHub Pages) อาจบันทึกข้อมูลการเข้าถึงมาตรฐาน เช่น IP address เพื่อความปลอดภัยของระบบ ตามนโยบายของผู้ให้บริการนั้น
- **ฟอนต์จาก Google Fonts** — หน้าเว็บโหลดฟอนต์จาก Google Fonts ซึ่งอาจมีการส่งคำขอไปยังเซิร์ฟเวอร์ของ Google ตามนโยบายความเป็นส่วนตัวของ Google

## การวิเคราะห์การใช้งาน (Cloudflare Web Analytics)

เราใช้ **Cloudflare Web Analytics** เพื่อทำความเข้าใจภาพรวมการเข้าชมเว็บไซต์ เช่น จำนวนหน้าที่ถูกเปิดดูและประเทศที่เข้าถึง โดยมีหลักการดังนี้

- **ไม่ใช้คุกกี้** — ระบบนี้ทำงานด้วย beacon script ไม่ได้ตั้งคุกกี้บนอุปกรณ์ของคุณ
- **ไม่เก็บข้อมูลระบุตัวบุคคล** — ไม่มีการเก็บชื่อ อีเมล หรือข้อมูลที่สามารถระบุตัวตนของผู้ใช้รายบุคคลได้
- **ข้อมูลรวมเท่านั้น** — ข้อมูลที่ได้เป็นสถิติภาพรวมเพื่อปรับปรุงเว็บไซต์ ไม่ใช้เพื่อการโฆษณาหรือติดตามข้ามเว็บไซต์
- **ผู้ให้บริการ** — ข้อมูลถูกประมวลผลโดย Cloudflare ตาม [นโยบายความเป็นส่วนตัวของ Cloudflare](https://www.cloudflare.com/privacypolicy/)

การวิเคราะห์นี้เปิดใช้เฉพาะบนเว็บไซต์จริง (thamdee.com) ไม่ทำงานในสภาพแวดล้อมพัฒนาบนเครื่องของเรา

## สิ่งที่เราไม่ทำ

- ไม่ขายหรือแบ่งปันข้อมูลของคุณให้บุคคลที่สามเพื่อการตลาด
- ไม่ใช้คุกกี้ติดตามพฤติกรรมเพื่อการโฆษณา
- ไม่เก็บข้อมูลเกินความจำเป็น

## สิทธิของคุณตาม PDPA

คุณมีสิทธิขอเข้าถึง แก้ไข หรือลบข้อมูลส่วนบุคคลที่เกี่ยวข้องกับคุณ (เช่น อีเมลที่เคยส่งถึงเรา) โดยติดต่อได้ที่ burapatis@gmail.com เราจะดำเนินการภายในระยะเวลาที่กฎหมายกำหนด

## การเปลี่ยนแปลงนโยบาย

หากมีการเพิ่มระบบใหม่ที่เกี่ยวข้องกับข้อมูลส่วนบุคคล เช่น ระบบสมาชิกหรือจดหมายข่าว เราจะปรับปรุงนโยบายฉบับนี้และระบุวันที่อัปเดตไว้อย่างชัดเจน
